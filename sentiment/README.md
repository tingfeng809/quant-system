## ⚠️ 预警规则

### 预警级别

| 级别 | 条件 | 响应时间 |
|------|------|---------|
| **🔴 红色** | 重大利好/利空 (|score| > 0.8) | 1 分钟 |
| **🟠 橙色** | 重要事件 (|score| > 0.6) | 3 分钟 |
| **🟡 黄色** | 一般事件 (|score| > 0.4) | 10 分钟 |
| **🔵 蓝色** | 关注事件 (|score| > 0.2) | 30 分钟 |

### 热度计算

```python
热度 = 情感分数 × 来源权重 × 传播速度 × 影响范围

- 传播速度：1 小时内转发/评论数
- 影响范围：涉及股票市值/行业重要性
```

### 触发条件

```python
# 单一事件触发
if abs(情感分数) > 0.7:
    触发预警()

# 多事件叠加
if 同一股票 1 小时内 3 条以上利空:
    触发红色预警()

# 行业联动
if 同一行业 5 家公司 同时出现利空:
    触发行业预警()
```

## 🧪 回测验证

### 舆情指数 vs K 线

```python
# 构建舆情指数
舆情指数 = Σ(情感分数 × 来源权重 × 时间衰减)

# 回测策略
if 舆情指数 > 阈值:
    买入()
elif 舆情指数 < -阈值:
    卖出()

# 验证指标
- 胜率
- 收益率
- 最大回撤
- 夏普比率
```

### 有效性验证

| 指标 | 目标 | 说明 |
|------|------|------|
| 胜率 | > 55% | 舆情方向与股价方向一致 |
